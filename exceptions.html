Exception handling

Class exception_list

  
namespace std {

  class exception_list : public exception
  {
    public:
      typedef unspecified iterator;
  
      size_t size() const noexcept;
      iterator begin() const noexcept;
      iterator end() const noexcept;

      virtual const char* what() const noexcept;
  };
}

The class exception_list owns a sequence of exception_ptr objects. The pParallel algorithms may use the exception_list to communicate uncaught exceptions encountered during parallel execution to the caller of the algorithm.

The type exception_list::iterator is a constant iterator that meets the requirements of ForwardIterator.

size_t size() const noexcept; The number of exception_ptr objects contained within the exception_list. Constant time. iterator begin() const noexcept; An iterator referring to the first exception_ptr object contained within the exception_list. iterator end() const noexcept; An iterator that is the past-the-end value of the owned sequence. virtual const char* what() const noexcept; An implementation-defined NTBS.
